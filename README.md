## INSTRUCTIONS

1. Make sure Node.js is installed on your system.

2. Clone the repo.

2. Run the command 'npm install http-server -g'.

3. Navigate to the repo and run the command 'http-server'.

4. Go to your browser and type 'localhost:8080'.
